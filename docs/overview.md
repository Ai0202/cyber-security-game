# CyberGuardians — 体験型セキュリティ教育ゲーム

## 概要

CyberGuardians は、「攻撃者の目線で学ぶ、サイバーセキュリティ体験学習」をコンセプトとした体験型の情報セキュリティ教育ゲームである。従来の座学中心のセキュリティ研修では実感が湧きにくいという課題を解決するため、プレイヤーが**攻撃者（ATTACK SIDE）と防御者（DEFENSE SIDE）の両方の視点**を体験できるインタラクティブなゲームを提供する。

攻撃シナリオは Gemini API によって毎回動的に生成されるため、繰り返しプレイしても新鮮な体験が得られ、飽きにくい設計となっている。セキュリティトピックごとに最適なゲーム形式を採用し、リアリティのある体験を通じて実践的なセキュリティリテラシーを身につけることを目指す。

## ターゲットユーザー

### メインターゲット
- 一般の会社員（IT部門以外を含む）
- セキュリティ研修の受講者

### サブターゲット
- 主婦・主夫（SNS経由の詐欺被害が増加傾向）
- 学生（SNSからの個人情報流出リスク）
- 情報セキュリティに興味はあるが専門知識がない方全般

## 技術スタック

- フロントエンド: Next.js (App Router) + TypeScript + Tailwind CSS
- アニメーション: Framer Motion
- AI: Google Gemini API（シナリオ動的生成・AIフィードバック・チャット対話）
- データベース: なし（MVP段階ではクライアントサイドのみ、状態はセッション内で管理）
- デプロイ: Google Cloud Run

## コンセプト

- **「攻撃を知ることが、最大の防御になる」** をテーマに、攻撃者の手口を体験して防御力を高める
- プレイヤーは「シャドウ」というハッカーキャラクターを操作して企業への侵入を試みる（攻撃側）
- 攻撃体験の後、防御側の視点も体験し、対策方法を学ぶ
- 各セキュリティトピックに最適なゲーム形式を採用（1トピック＝1ステージ、形式は異なる）
- サイバーパンク風のかっこいいUI（ダーク基調、ネオンカラーのアクセント）で没入感を演出
- Gemini APIを活用し、毎回異なるシナリオで飽きさせない

## スコープ

### MVP（本プロジェクトの対象）

- 4ステージ構成のプロトタイプ（攻撃側）
  - Stage 1: ショルダーハッキング（覗き見で情報を盗め）
  - Stage 2: パスワードクラッキング（弱いパスワードを突破せよ）
  - Stage 3: フィッシング攻撃（偽メールで騙せ）
  - Stage 4: ランサムウェア侵攻（サーバーを暗号化せよ）
- フィーチャーステージ: OPERATION: RANSOMWARE（4ステージを攻撃チェーンとして連続体験）
- 各ステージにスコアリング＋難易度表示
- ステージクリア後の結果画面 + AIによるフィードバック・解説
- ステージ選択ハブ画面（タブ切り替え: ステージ / キャラ / 体験デモ）

### 将来の拡張（MVP対象外）

- DEFENSE SIDE（防御者体験）ステージ群
- ユーザー認証・プロフィール
- ランキング機能
- 実績バッジシステム
- 他ユーザーの情報閲覧
- ステージの追加（SNS詐欺、Wi-Fiスニッフィング等）
